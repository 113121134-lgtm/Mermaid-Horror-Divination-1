# 🧜‍♀️ 深海之瞳：人魚恐怖占卜 (The Mermaid's Whispers)

![Theme](https://img.shields.io/badge/Theme-Horror%20%7C%20Occult-darkblue)
![Platform](https://img.shields.io/badge/Platform-GitHub%20Pages-green)

這是一個基於 **GitHub Pages** 建立的心理恐怖風格占卜網頁。結合了深海恐懼（Thalassophobia）與人魚傳說中的誘惑與惡意。

## 🌙 專案簡介
使用者將進入一個充滿壓抑感的深海介面，透過「獻祭祭品」的儀式感，潛入萬米深海，獲取來自人魚的恐怖預言。

## 🕯️ 進階功能特色
- **儀式啟動介面**: 要求使用者輸入特定的「文字祭品」，增強互動性與心理暗示。
- **動態下潛特效**: 模擬水壓增加、光線消失的動態過程。
- **3D 翻轉卡牌**: 透過 CSS 3D 變換技術製作的占卜卡片，點擊即可揭示命運。
- **視覺故障藝術 (Glitch Art)**: 標題與介面會不定期產生視覺破碎效果。

## 🛠️ 技術架構
- **HTML5**: 結構與語義化。
- **CSS3**: 包含 Flexbox 佈局、3D 翻轉動畫、Keyframes 視覺特效。
- **Vanilla JavaScript**: 處理隨機占卜邏輯、場景切換與數值變動。

## 🚀 快速上線指南
1. **上傳檔案**: 將 `index.html`, `style.css`, `script.js` 上傳至 GitHub。
2. **啟動 Pages**: 
   - 進入 **Settings** -> **Pages**。
   - 選擇 **Branch: main** 並按下 **Save**。
3. **訪問網頁**: 約一分鐘後即可透過專屬網址開啟。
